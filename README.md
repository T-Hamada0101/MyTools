# MyTools

日常の作業を効率化するために作成した自作の便利ツール群をまとめたリポジトリです。
主に C# で開発された各種デスクトップアプリケーションやユーティリティが格納されています。

完全に自分専用のツールなので、使い勝手は悪いかもしれません。

## 収録ツール一覧

- **EncodeAuto**
  動画や音声のエンコードを自動化・支援するツールです。エンコードにはNVEncCを主に使用しGPUで高速に並列エンコードできます。
  モバイル端末用に大量の動画ファイルを小さくするために作成しました。

- **yt-dlp_loader**
  `yt-dlp` を活用して動画を簡単にダウンロードするためのローダー（GUI）です。

- **MuteVideoTwimgCom**
  Twitter (X) の動画コンテンツをミュート化、あるいは処理するブラウザ機能拡張です。動画を再生するたびに音量がMAXで再生されるのを防ぐために作成しました。

- **HDD_Info**
  ハードディスクのステータスやストレージ情報を取得・表示するツールです。WMIから取得しています。

- **RegEditTools**
  Windowsレジストリの操作や確認を補助するツールです。

- **MovieCut**
  動画ファイルの無劣化カットや編集処理を行うffmpegラッパーです。

- **MyExcelReader**
  Excelファイルの読み込みやデータ抽出を簡略化するツールです。

- **_Copilot**
  AI アシスタント関連のテストや実験的なコードが含まれるプロジェクトです。まだ中身はありません。

## 開発環境

- Windows 11 / 10
- Visual Studio 2022
- C# (.NET)
